Floade v0.1.0

最初版 AI 助手，主打「隨手記」。目標是讓你隨手記下臨時想法，不用像 Notion 那樣特別開一頁。先記、再整理、隨時可用。

FAQ

1) Floade 是什麼？
- 目前主打「隨手記」的筆記工具，未來會朝「祕書」方向進化。

2) 特色是什麼？
- 重點是隨手記，內容可自動整理（前提：你已設定 API Key / Model），也能手動整理。

3) 資料會送到哪裡？
- 目前只會送到 OpenRouter，而且只有你按下整理時才會送出。

4) 沒有設定 API Key 可以用嗎？
- 可以，筆記功能不受影響；只是自動整理會停用。

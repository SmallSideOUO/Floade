# WorkStatus

以「工作狀態」為核心的輕量工具，讓個人與小團隊清楚追蹤現在、接下來、與未來計畫。

## 為誰設計
- 個人工作管理者
- 小型團隊與自由工作者
- 想要更清楚對外同步狀態的人

## 你可以用它做什麼
- 聚焦「現在正在做什麼」
- 快速對外同步「目前進度」
- 用最小成本維持一致的工作節奏

## 下載與安裝
- 請至本倉庫的 Releases 下載最新版本
- 下載後直接執行 `WorkStatus.exe`

## 快速開始
1. 執行 `WorkStatus.exe`
2. 依畫面指示建立第一筆狀態
3. 完成後即可開始使用
